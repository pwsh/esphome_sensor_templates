# LD2410 mmWave Presence

HiLink LD2410 24GHz mmWave radar presence sensor over UART. Exposes occupancy plus moving/still target flags and distances. Owns its own uart: bus.

**Platforms:** `esp32` `esp32s2` `esp32s3` `esp32c3` `esp32c6`

**Requires:** none

## Entities

| Domain | Name |
|---|---|
| `binary_sensor` | Presence |
| `binary_sensor` | Moving Target |
| `binary_sensor` | Still Target |
| `sensor` | Moving Distance |
| `sensor` | Still Distance |

## Usage

Local include (repo checked out beside your config):

```yaml
packages:
  ld2410: !include
    file: esphome_sensor_templates/templates/presence/ld2410.yaml
    vars: { st_name_prefix: "", st_ld2410_uart_tx_pin: <value>, st_ld2410_uart_rx_pin: <value> }
```

Remote include (pulled straight from GitHub):

```yaml
packages:
  ld2410: github://OWNER/esphome_sensor_templates/templates/presence/ld2410.yaml@main
```

> Replace `OWNER` with the GitHub owner/repo that hosts this library.

## Variables

| Variable | Default | Description |
|---|---|---|
| st_name_prefix | `""` | Prefix prepended to every entity name |
| st_disabled_by_default | `false` | Ship the entities disabled in HA |
| st_internal | `false` | Hide the entities from HA/web entirely |
| st_state_class | `measurement` | state_class for the distance sensors; set "" to opt out of HA long-term statistics |
| **st_ld2410_uart_tx_pin** | **(required)** | GPIO wired to the sensor RX pin (ESP TX) |
| **st_ld2410_uart_rx_pin** | **(required)** | GPIO wired to the sensor TX pin (ESP RX) |

## Notes

- The LD2410 uses a fixed 256000 baud UART (parity NONE, 1 stop bit). Use hardware UART pins. ESP32 has 3 hardware UARTs, S2/S3 have 2-3, C3/C6 have 2 - one is usually taken by the logger (UART0), so keep the logger on its default pins or drop its baud/hardware_uart if you run out.
- This preset exposes only the core presence readings. The full tuning surface - per-gate move/still sensitivity, detection timeout, bluetooth, engineering mode, calibration buttons - is available by adding the ld2410 number:/select:/switch:/button: platforms with ld2410_id: st_ld2410, or by !extend st_ld2410.
